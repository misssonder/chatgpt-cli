#  ChatGPT Client
## Install
```shell
cargo install --git https://github.com/misssonder/chatgpt-cli
```
## Usage
```shell
Simple ChatGPT command line

Usage: chatgpt --api-key <API_KEY>

Options:
  -a, --api-key <API_KEY>  Api key of the openai support
  -h, --help               Print help
  -V, --version            Print version
```
